Una vez los recursos sean asignados y recibas el correo con las credenciales de ingreso, debes realizar la conexión según el sistema operativo que utilices: conexión desde Windows o conexión desde Linux o Mac IOS

Conexión desde Windows

Debes instalar un cliente SSH en Windows, te recomendamos Mobaxterm.

  • Se debe crear una sesión donde el host (servidor) debe ser: hypatia.uniandes.edu.co y el usuario corresponde al usuario Uniandes habilitado.
  • Posteriormente se solicitará la contraseña, la cual es la misma de la cuenta Uniandes.
  • Si es la primera vez que utilizas Mobaxterm te preguntará si deseas guardar la contraseña, si te conectas desde un computador personal o una red conocida te recomendamos guardarla para mayor rapidez en futuras conexiones.

Conexión desde Linux o Mac IOS

La conexión para los sistemas operativos Linux y Mac puede realizarse desde el intérprete de comandos shell de la siguiente manera: 

  • Establecer la conexión con el comando: ssh <user>@hypatia.uniandes.edu.co</user>  
  • Aparecerá una confirmación de la conexión, la cual debes aceptar escribiendo: yes
  • Por último, ingresar la contraseña.
